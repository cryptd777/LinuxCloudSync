## ğŸš€ How to Use New Features

### Creating a Profile

1. Go to **Sync** tab
2. Fill in:
   - Remote: `gdrive:`
   - Local Folder: `/home/user/Documents`
   - Choose sync mode
3. Set any advanced options (optional)
4. Click **"Save Profile"**
5. Name it: "My Documents"

**Done!** Now you can load it anytime from Profiles tab.

### Using Bandwidth Limit

1. Go to **Advanced** tab
2. Enter bandwidth limit:
   - `1M` = 1 MB/s
   - `500K` = 500 KB/s
   - `10M` = 10 MB/s
3. Start sync as normal

### Using Dry Run

1. Go to **Advanced** tab
2. Toggle **"Dry Run"** switch ON
3. Go to **Sync** tab
4. Click "Start Sync"
5. Watch log - shows what WOULD happen
6. No files actually transferred!

### Fixing Bisync Errors

**If you see this error:**
```
ERROR: Bisync critical error: cannot find prior Path1 or Path2 listings
ERROR: Bisync aborted
```

**Solution:**
1. Go to **Sync** tab
2. Click **"Force Resync"** button (orange)
3. Confirm the prompt
4. Wait for completion
5. Click "Start Sync" normally

**That's it!** Error fixed.
